## 🎨 Key Concepts Explained

### Glass Morphism
A modern design trend using frosted glass effects with backdrop blur. Creates depth while maintaining modern aesthetics.

**Where it's used:**
- Card backgrounds
- Dialog/modals
- Button backgrounds

**Reference:** DARK_MODE_CSS_IMPLEMENTATION.md

---

### Glow Effects
Subtle text shadows and box shadows that make text and elements "glow" with the primary color.

**Where it's used:**
- Headlines and important text
- Icon badges
- Primary buttons

**Reference:** PAGE_BY_PAGE_ENHANCEMENTS.md

---

### Visual Hierarchy
Using color, size, and position to guide user attention to important elements.

**Improved in:**
- Settings page (color-coded groups)
- Home page (enhanced hero card)
- Transaction list (card separation)

**Reference:** BEFORE_AFTER_COMPARISON.md
