🎯 WHAT IS THE SAVINGS FEATURE?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  A new transaction type (savings) that:

  ✅ Decreases wallet balance (like expenses)
  ❌ Does NOT count as an expense (unlike expenses)
  ❌ Does NOT count toward budget (unlike expenses)
  ✅ Automatically increases linked goal progress (unique!)
  ✅ Optional goal linking (goal_id field)

  Example: User saves 500k toward "Du lịch" goal
  → Wallet: 2M → 1.5M (balance decreases)
  → Goal: 1M → 1.5M (progress increases)
  → Expense stats: Unchanged (savings excluded)
  → Budget: Unchanged (savings excluded)
